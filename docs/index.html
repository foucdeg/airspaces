Self-host an Airspace

The easiest way to self-host an Airspace is to run the Docker image:

docker run -d -p "9000:9000" -p "49003:49003/udp" foucdeg/airspaces

The Airspaces API will be available on port 9000 and the X-Plane port is 49003. The client will soon be integrated in the Docker image; for now, you can build it from its Github repository.

If you cannot use Docker, you can run the Node.js backend and serve the ReactJS client in your own preferred way, as they are both open source.
